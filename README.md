# Game boy

![image](https://github.com/n1md7/game-boy/assets/6734058/fbc88e9e-bc1d-4c7c-a980-04b4328ee190)



Technologies used

- Vite
- Vitest
- Typescript
- Electron

Features

- FPS controls (WASD)
- Orbit controls (Mouse)
- Object collisions

## How to run

```bash
# Browser
$ npm run dev

# Build - Transpile to JS
$ npm run build


# Electron
$ npm run electron
# or
$ npm run start

# Build binaries
$ npm run make
$ npm run package
```
