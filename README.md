# Game boy

Technologies used

- Vite
- Vitest
- Typescript
- Electron

Features

- FPS controls (WASD)
- Orbit controls (Mouse)
- Object collisions

## How to run

```bash
# Browser
$ npm run dev

# Build - Transpile to JS
$ npm run build


# Electron
$ npm run electron
# or
$ npm run start

# Build binaries
$ npm run make
$ npm run package
```
